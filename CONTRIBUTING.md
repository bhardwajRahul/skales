# Contributing to Skales

Thanks for your interest in contributing!

## How to Contribute

1. **Bug Reports**: Open an issue with your OS, Skales version, provider/model, and steps to reproduce.

2. **Translations**: Skales supports 12 languages. Translation files are in apps/web/src/locales/. Copy en.json and translate the values.

3. **Custom Skills**: Build skills and share them via Discover Feed or the Agent Skills format (SKILL.md).

4. **Bug Fixes**: Fork, create a branch, fix, open a PR.

## What We Accept

- Bug fixes
- Translations
- Documentation improvements
- Non-commercial feature contributions

## What We Don't Accept

- Changes to licensing
- Commercial feature additions without discussion
